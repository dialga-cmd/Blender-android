说明:
1 lib-android_arm64 		存储的是编译Blender库需要的第三方库文件，包括库文件和头文件，后续会上传各个库文件的源文件。
目前为了简单处理，在blender库中硬编码了不同库文件的路径。有需要的话，可以将 D:/FreeProjects/Blender/Blender 替换成 自己需要的文件夹路径。
2 Utilities-android_arm64 	存储的是编译Blender库需要的辅助工具，包括生成DNA文件项目、生成RNA文件项目、其它软件。
3 Blender-android_arm64 	存储的是Blender库文件，初始版本是从blender仓库拉取的blender-v3.6-release分支，并基于这个分支，同步
本地的修改，后续在该仓库继续修改。
4 APP-android_arm64			存储的是编译APP需要的源文件和资源文件。

流程:
1）下载lib-android_arm64、Utilities-android_arm64、Blender-android_arm64、APP-android_arm64四个仓库的源码
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2）【Android Studio项目】第一次编译 【makeblender】 直到生成 libmakesdna.a
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3）【Android Studio项目】编译 【MakeSDNA】
4）复制 Blender-android_arm64 下的 makesdna 文件夹 到 /data/data/com.epai.makesdna/
5）运行 MakeSDNA
6）复制/data/data/com.epai.makesdna/下的/data/data/com.epai.makesdna/dna.c、/data/data/com.epai.makesdna/dna_type_offsets.h、/data/data/com.epai.makesdna/dna_verify.c文件到任意位置
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
7）注释 D:/FreeProjects/Blender/Blender/Blender-android_arm64/文件夹下的
add_custom_command(
  OUTPUT
    ${CMAKE_CURRENT_BINARY_DIR}/dna.c
    ${CMAKE_CURRENT_BINARY_DIR}/dna_type_offsets.h
    ${CMAKE_CURRENT_BINARY_DIR}/dna_verify.c
  COMMAND
    "$<TARGET_FILE:makesdna>"
    ${CMAKE_CURRENT_BINARY_DIR}/dna.c
    ${CMAKE_CURRENT_BINARY_DIR}/dna_type_offsets.h
    ${CMAKE_CURRENT_BINARY_DIR}/dna_verify.c
    ${CMAKE_SOURCE_DIR}/source/blender/makesdna/
  DEPENDS makesdna
)
8）复制 D:\FreeProjects\Blender\Blender\Utilities-android_arm64\Utilities文件夹下的msgfmt.exe、datatoc.exe等 文件
到 D:/FreeProjects/Blender/Blender/Utilities-android_arm64/MakeBlender/app/.cxx/cmake/release/arm64-v8a/source/creator/ 文件夹
复制 D:\FreeProjects\Blender\Blender\Utilities-android_arm64\Utilities文件夹下的msgfmt.exe、datatoc.exe等 文件
到 D:/FreeProjects/Blender/Blender/Utilities-android_arm64/MakeBlender/app/.cxx/cmake/release/arm64-v8a/bin/ 文件夹
9）替换 dna_verify.c 中的 "/data/data/com.epai.makesdna/makesdna" 为 "D:/FreeProjects/Blender/Blender/Blender-android_arm64/source/blender/makesdna"
10）将上面三个文件复制到 D:/FreeProjects/Blender/Blender/Utilities-android_arm64/MakeBlender/app/.cxx/cmake/release/arm64-v8a/source/blender/makesdna/intern 文件夹
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
11）【Android Studio项目】第二次编译 【makeblender】 直到生成 libmakesrna.a
12）【Android Studio项目】编译 【MakeSRNA】
13）运行 MakeSRNA
14）复制/data/data/com.epai.makesrna/*.c *.h到任意位置
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
15）注释 D:/FreeProjects/Blender/Blender/Blender-android_arm64/文件夹下的
add_custom_command(
  OUTPUT ${GENSRC}
  COMMAND "$<TARGET_FILE:makesrna>" ${CMAKE_CURRENT_BINARY_DIR}/ ${CMAKE_CURRENT_BINARY_DIR}/../
  DEPENDS makesrna
)
16）复制 /data/data/com.epai.makesrna/*.c *.h 到 D:\FreeProjects\Blender\Blender\Utilities-android_arm64\MakeBlender\app\.cxx\cmake\release\arm64-v8a\source\blender\makesrna\intern文件夹
17）复制 /data/data/com.epai.makesrna/*.h 到 D:\FreeProjects\Blender\Blender\Blender-android_arm64\source\blender\makesrna\文件夹
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
18）【Android Studio项目】第三次编译 【makeblender】 直到完成所有库编译
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
19）【Android Studio项目】编译 【OBlender】
20）运行生成的APP
