Description:
1 lib-android_arm64 stores third-party library files required to compile the Blender library, including library files and header files. The source files of each library file will be uploaded later.
At present, for simple processing, the paths of different library files are hard-coded in the blender library. If necessary, you can replace D:/FreeProjects/Blender/Blender with the folder path you need.
2 Utilities-android_arm64 stores auxiliary tools required to compile the Blender library, including generating DNA file projects, generating RNA file projects, and other software.
3 Blender-android_arm64 stores Blender library files. The initial version is the blender-v3.6-release branch pulled from the blender warehouse, and based on this branch, synchronizes
local modifications, and continues to modify in the warehouse later.
4 APP-android_arm64 stores the source files and resource files required to compile the APP.

5) Run MakeSDNA
6) Copy /data/data/com.epai.makesdna/dna.c, /data/data/com.epai.makesdna/dna_type_offsets.h, /data/data/com.epai.makesdna/dna_verify.c files under /data/data/com.epai.makesdna/ to any location

"$<TARGET_FILE:makesdna>"
${CMAKE_CURRENT_BINARY_DIR}/dna.c

${CMAKE_CURRENT_BINARY_DIR}/dna_verify.c
${CMAKE_SOURCE_DIR}/source/blender/makesdna/
